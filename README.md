## Pourquoi ce site ?

Le Bazar du Feuch est simplement une vitrine de mes projets.

Certains sont déjà disponibles.

D'autres avancent tranquillement.

Si une création vous amuse, vous intrigue ou vous aide, vous pouvez :

* la partager ;
* l'essayer ;
* l'acheter ;
* ou soutenir le laboratoire.

## À propos

Je m'appelle Benoît Lubert.

Je crée des livres, des jeux, des applications et quelques univers imaginaires depuis plus longtemps qu'il n'est raisonnable de l'avouer.

Le Feuch Institute n'existe pas officiellement.

Mais nous poursuivons nos recherches.

Parce qu'il reste encore beaucoup de choses étranges à observer.
